Member Management System (MMS)
Java Swing + Text Interface

How to run (simple):

1. Using command-line javac/java
   - Create folders:
        MemberManagementSystem/
        MemberManagementSystem/src/
        MemberManagementSystem/data/
   - Put all .java files into src/
   - Put members.txt into data/ (sample below)
   - Compile:
        javac -d bin src/*.java
   - Run:
        java -cp bin Main

2. Notes:
   - The GUI uses Java Swing.
   - Data is stored in data/members.txt as CSV (id,name,type,fee).
   - The program auto-saves after operations.
   - Fill your name and student ID into Individual_Report_MMS.docx before submission.
